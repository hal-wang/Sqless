# 简介

通过权限配置和身份认证，使客户端即能随意操作数据库，又确保数据库安全可控，并且在客户端用户仅能操作允许操作的数据。

开发者在简单配置服务端后，可简单安全的操作数据库，将大部分精力放在客户端的业务逻辑。

# 服务端

服务端配置好数据库读写权限。接口需接受 SqlessRequest派生类作为参数，比如 WebApi 中Controller下的Action。

服务端也可配置身份认证，设定身份认证方式和身份认证数据所在表，当客户端调用接口时，验证认证信息。

# 客户端

调用接口方法与普通接口一样，但传参数是 SqlessRequest的派生类。

此外，Sqless封装了 WebApi 的调用，使用非常简单。
